# Changelog

All notable changes to this project are documented here. The project follows
[Semantic Versioning](https://semver.org/).

## Unreleased

### Added

- Strict, versioned `sse-run-v1` YAML configurations
- `sse validate`, `sse run`, and `sse inspect` commands
- Versioned manifests, summaries, measurements, and per-chain artifacts
- Chain-level deterministic restart support
- Integrated autocorrelation time, effective sample size, and split R-hat
  diagnostics
- Physics-first mdBook guide and GitHub Pages deployment
- Tagged release workflow for Linux, macOS, and Windows binaries

## 0.1.0 - 2026-07-19

### Added

- TFIM and long-range Rydberg SSE decompositions
- TFIM linked-cluster and Rydberg world-line updates
- Deterministic independent-chain TFIM execution
- Dense small-system Rydberg validation example
- Rust API documentation and Apache-2.0 licensing
